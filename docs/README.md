# PEAK CTI Dashboard

This folder contains the interactive PEAK CTI analytics dashboard, served via GitHub Pages.

## Viewing the Dashboard

**Live:** [https://apps-dfir.github.io/Security-Operations/](https://apps-dfir.github.io/Security-Operations/)

## Features

- Reports timeline
- IOC distribution by type
- MITRE technique frequency
- Common IOCs across reports
- Recent reports list

## Updates

The dashboard is automatically regenerated when new reports are processed.
