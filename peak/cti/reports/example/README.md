# example/ - Sample Reports

This directory contains example reports demonstrating PEAK CTI output.

## Purpose

These samples help you understand:
- Report structure and formatting
- How IOCs are extracted and scored
- MITRE ATT&CK technique mapping
- OCR extraction results

## Using Examples as Reference

Review these reports before submitting your first issue to understand what the system produces. Compare the source articles with the generated output to see how extraction works.
