# reports/ - Generated CTI Reports

All CTI reports generated by PEAK CTI are stored here after PR merge.

## Report Naming

```
issue-{number}_{title-slug}_consolidated_{timestamp}.md
```

## Searching Reports

```bash
# Find reports mentioning a hash
grep -r "abc123" reports/

# Find reports with specific domain
grep -r "evil.com" reports/

# List recent reports
ls -lt reports/ | head -10
```
