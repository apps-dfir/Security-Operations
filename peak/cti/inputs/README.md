# inputs/ - Source File Uploads

Upload PDF and DOCX files here for processing.

## How to Upload

1. Add file to this directory via GitHub or git
2. Reference in issue as `inputs/filename.pdf`

## Supported Types

- `.pdf` - PDF documents
- `.docx` - Word documents
- `.txt` - Plain text
- `.md` - Markdown

## Example

```markdown
### Sources (up to 5)

1. inputs/cisa-advisory.pdf
2. inputs/malware-analysis.docx
```
